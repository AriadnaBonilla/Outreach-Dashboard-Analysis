Methodology
How This Dashboard Works
Data Sources
DG Action Plans 8 MU + 4 LOB files
One Excel per Market Unit (UKI, BeNeLux, Spain, Southern Europe, Nordics, France, MEA South, Italy) and one per Line of Business (BTP, F&S, HCM&CX, SCM). Each file contains the planned campaigns with their Sequence Links to Outreach, WBS codes, target pipeline, industry, executor and status.
Outreach Reports Sequence_Stats sheet
One row per sequence with: Prospects Total/Active/Finished/Replied/Opened/Opted Out, Emails Deliveries/Opens/Clicks/Replies. The Owner column is used as the SDE field. Updated manually when a new Outreach export is dropped in the folder.
Curators Request List SharePoint MS List
Every campaign the DG Curators team has supported. Contains WBS code + Region. A campaign is marked ✦ Curated when its (WBS code, MU) pair exists in this list.
Global Cloud Pipeline Analyzer SAC export
Opportunities and ACV per campaign (keyed by Opp Campaign ID = WBS code). Contains Region Lvl 2, Closing Quarter, Opp Phase and ACV in EUR (converted to kEUR in this dashboard).
How We Connect the Files
1
DG Plan → Outreach
The Sequence Link column in each DG Plan contains a URL like outreach.io/sequences/68090. We extract the numeric ID and match it to the Sequence ID column in the Outreach Stats sheet. Only campaigns with a valid match appear in this dashboard.
2
DG Plan → Curators
Match on (WBS Code + MU). Both fields must match exactly. MU names are normalised: "CS Nordic" → "CS Nordics", "CS Southern Europe" in files → "CS Spain", and "CS Türkiye / Israel / Portugal / Greece & Cyprus" → "CS Southern Europe".
3
DG Plan → Pipeline
Match on WBS Code = Opp Campaign ID. Pipeline is only assigned to a campaign if it also has Prospects > 0 in Outreach (prevents empty sequences claiming pipeline). Pipeline ACV is deduplicated by WBS — if multiple campaigns share a WBS, the ACV is counted once in the totals.
!
Deduplication
A sequence that appears in both a MU Plan and a LOB Plan is only counted once per MU. The source tag ("mu" or "lob") is assigned based on which file it first appeared in. The Top Performing By MU card shows only MU-plan sequences; By LOB shows only LOB-plan sequences.
Metrics Explained
Reply Rate % = Prospects Replied ÷ Prospects Total × 100
Unique prospects who replied at least once. High Performers threshold: ≥5% with ≥15 prospects.
Open Rate % = Prospects Opened ÷ Prospects Total × 100
Prospects who opened at least one email. Calculated per-prospect, not per email.
Click Rate % = Emails Clicks ÷ Emails Deliveries × 100
Per-email click rate (not per-prospect). Includes all clicks across all emails delivered.
Opt-Out Rate % = Prospects Opted Out ÷ Prospects Total × 100
Prospects who unsubscribed. Monitor this — high opt-out may indicate wrong targeting.
Pipeline vs Target % = Actual ACV kEUR ÷ Target Pipeline kEUR × 100
Target comes from DG Plan "Target Pipeline - value kEUR" column (converted from EUR). Actual comes from the Pipeline Analyzer. Only campaigns with both pipeline data and Outreach stats contribute.
Curators Coverage % = Curated campaigns ÷ Total sequences × 100
Of all DG Plan campaigns matched to Outreach, what % did the Curators team support. Target for 2026: 35%.
Dashboard Sections
Outreach Metrics (8 KPI cards)
Global aggregates across all matched campaigns in the current filter. Avg rates are weighted by prospects (not a simple average of %).
Campaigns We Support (green banner)
Campaigns supported by the DG Curators team. Opps and ACV shown are from the Pipeline file, deduplicated by WBS. The mini bar chart shows curated campaign count per MU.
Curators Coverage by MU (stacked bar)
Green = curated by us. Grey = not curated. % label at the end of each bar shows coverage rate. Hover for exact numbers.
Top Performing Sequences (3 cards)
All EMEA: top 15 across all sources, responds to global filters. By MU: always uses the full MU DG Plan data (unaffected by global filters) — select a MU to see its top 15. By LOB: same but for LOB DG Plans. Min 15 prospects. Click any row to open the sequence in Outreach.
SDE Breakdown
SDE = the Outreach sequence Owner (not the DG Plan Executor). Shows who is actually running the sequences and their engagement rates. Min 2 sequences to appear.
Full Campaign Data table
All matched campaigns. Chip ✦ Curated filters to campaigns we supported. Chip ⬥ Pipeline filters to campaigns with pipeline data. Click ↗ to open in Outreach. Row turns green for curated campaigns.
Known Data Gaps
CS France — 0 sequences matched
The France DG Plan has no Sequence Links or Sequence IDs filled in. The Outreach column is empty for all rows.
BeNeLux & MEA South — partial match
Some campaigns have Sequence Links that point to sequences not present in the current Outreach export. Refresh the Outreach Reports file to pick up missing sequences.
Shared WBS codes
Some WBS codes (e.g. CRM-WE25-S4H-UK06_BHNR) are used by 40+ campaigns. Pipeline ACV for those WBS codes cannot be attributed to a single campaign — it is shown at the WBS level and deduplicated in the totals.
